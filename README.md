
# 🏥 Health Chatbot for FAQs

## 📌 Problem Statement

In today’s digital age, people often rely on the internet and social media for health-related information. However, much of this information is:

* Unverified
* Misleading
* Inaccurate

This leads to:

* Incorrect self-diagnosis
* Delayed medical treatment
* Increased health risks
* Anxiety and confusion

---

## 💡 Solution

The **Health Chatbot** is an AI-powered web application that provides:

* ✅ Verified answers to common health questions
* 🤖 Instant chatbot-based responses
* 🩺 Suggestions to consult doctors when needed
* 📚 Reliable and structured health information

---

## 🚀 Features

* 💬 Chat-based user interface
* ⚡ Fast and real-time responses
* 🧠 Basic AI/NLP logic for answering queries
* 📱 Responsive frontend design
* 🔗 Integration between frontend and backend

## ⚙️ How to Run the Project

### 🔹 Step 1: Clone the Repository

```bash
git clone https://github.com/pandiansaravana497-cloud/health-chatbot.git
cd health-chatbot
```

### 🔹 Step 2: Setup Backend

```bash
cd backend
pip install -r requirements.txt
python app.py
```

### 🔹 Step 3: Run Frontend

* Open `frontend/index.html` in your browser
